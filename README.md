# AutoPrinter
Excel add-in, replace to a printing VBA macro.

VBA による印刷マクロを Excel アドイン形式の GUI に置き換えるもの。

## インストール
- ```InstallerAutoPrinterExcelAddIn.zip``` を展開して ```setup.exe``` を実行してください。
